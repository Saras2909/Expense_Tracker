# 💰 Expense Tracker

A CLI Expense Tracker built with Python that helps users organize and manage their expenses using categories. All data is stored persistently in a JSON file.

## Features

- Create custom expense categories
- Add expenses to any category
- View all expenses in a category
- Modify existing expenses
  - Add to an amount
  - Subtract from an amount
  - Rename an expense
  - Set a new amount
- Calculate the total expenses for a category
- Automatic data persistence using json
- Menu-driven command-line interface
